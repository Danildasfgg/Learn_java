Типы данных:
String- сохраняет текст, например "Hello". Строковые значения заключаются в двойные кавычки
int- хранит целые числа без десятичных знаков, например 123 или -123
float- хранит числа с плавающей точкой, с десятичными знаками, например 19,99 или -19,99
char- сохраняет отдельные символы, такие как 'a' или 'B'. Значения символов заключаются в одинарные кавычки
boolean- сохраняет значения с двумя состояниями: True или False

int myNum = 5;
float myFloatNum = 5.99f;
char myLetter = 'D';
boolean myBool = true;
String myText = "Hello";
